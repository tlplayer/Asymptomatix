# Asymptomatix

INSTALLATION

```sh
python3 -m venv env
source env/bin/activate
pip install -r requirments.txt
```

Point the FLASK_APP variable at the directory/file that holds the app
```sh
export FLASK_APP=flaskr
export FLASK_ENV=development
export FLASK
export FLASK_RUN_PORT=8008
```
Run the flask app
```sh
flask run
```
